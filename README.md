# NBA Stats Dashboard

An interactive NBA analytics dashboard built with Python and Streamlit for analyzing team performance metrics.

## Features

- Interactive team selection
- Team statistics table
- Wins and losses tracking
- Points per game analysis
- Rebounds and assists metrics
- Interactive data visualizations
- Streamlit dashboard interface

## Technologies Used

- Python
- Streamlit
- Pandas
- Plotly

## Dashboard Preview

This dashboard allows users to:

- View NBA team statistics
- Compare team performance
- Analyze offensive production
- Explore team metrics interactively
